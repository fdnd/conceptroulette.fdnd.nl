Jouw thema:


Neem de volgende stappen:

  1. Doe vooronderzoek en bepaal de elementen van jouw stijl. Gebruik screenshots of Pinterest om een verzameling te maken. Neem een collage op in de betreffende issue.
  2. Schets jouw concept. Bedenk welke interacties, animaties en geluidsondersteuning je nodig hebt. Neem je schets op in de betreffende issue.
  3. Maak een breakdown schets waarin je onderzoekt hoe je jouw concept met HTML, CSS en Javascript kunt implementeren. Neem ook jouw breakdown schets op in de betreffende issue.
  4. Implementeer jouw concept in HTML/CSS/JS en Sveltekit. Beschrijf in de betreffende issue hoe je de belangrijkste dingen voor elkaar krijgt.
  5. Test uitvoerig en beschrijf verschillen in performance in de betreffende issue
  6. Voer een pull-request uit waarin je de belangrijkste zaken uit bovenstaande issues opneemt. (nb. voer het PR nog niet door!)
  7. Publiceer jouw project via Vercel of Netlify (je selecteert dan de aparte branch).
  8. Laat je concept testen in een user-test tijdens de Design & Code review!
